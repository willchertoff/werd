# werd
Webpack, express, react, redux template

## Overview
Run a `webpack-dev-server` alongside a node express server.


## To run it locally
1. `$ npm install`
2. `$ npm run start`
3. visit `http://localhost:5002`

## Testing
1. For command line tests, `$ npm run test`

## Linting
1. To run the linter, `$ npm run lint`
